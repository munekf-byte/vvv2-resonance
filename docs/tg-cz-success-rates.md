# CZ（レミニセンス / 大喰いの利世）成立役別 実質成功抽選

## 成立役別の成功抽選確率

| 成立役 | レミニセンス（1〜7G） | 大喰いの利世（1〜7G） |
|---|---|---|
| リプレイ | 0.1%未満 | 26.56% |
| 押し順・斜めベル | 10.94% | 10.94% |
| 弱レア役 | 40.61% | 100% |
| 強レア役 | 100% | 100% |

## 補足ルール

- 最終ゲーム（8G目）に**弱レア役以上**を引けば**成功濃厚**
- **大喰いの利世**の場合、最終ゲームで**リプレイや斜めベル**でも**成功濃厚**へ格上げ
- CZ消化は8G間
- 大喰いの利世はレミニセンスより全体的に成功率が高い（特にリプレイで26.56%）
